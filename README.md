# Daly Family Tree

Interactive family tree for Joanne Daly & kindred — five generations, searchable, with desktop pan/zoom and a mobile layout.

## Live site

After GitHub Pages is enabled: **https://kieranjc.github.io/jo-family-tree/**

## Local preview

Open `index.html` in a browser, or run a simple server:

```bash
python3 -m http.server 8080
```

Then visit http://localhost:8080

## Files

| File | Purpose |
|------|---------|
| `index.html` | Main app (used by GitHub Pages) |
| `Family Tree.html` | Same app, original export filename |
| `Family Tree (standalone).html` | Bundler wrapper variant (not needed for Pages) |
